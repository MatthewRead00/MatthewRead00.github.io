Project information

  • Category: Front-end Development
  • Client: 45 Drives
  • Project date: April - August 2020
  • Project URL: www.45drives.com

Details:

While working for Protocase, I was tasked with maintaining the Protocase website, as well as the 45 Drives website. While working on 45drives.com, I reported to the Senior Web Developer for Protocase and 45 Drives, as well as 45 Drives' marketing manager. Throughout my time working for Protocase and 45 Drives, I was part of the company's marketing team.

For the 45 Drives website, I created and maintained various pages, including sitewide articles, weekly tech tip pages, the "About" pages, and some of the 45 Drives "Industries" pages, which goes over a list of industries that 45 Drives sells their projects to. I also created brand new pages for the site, such as the 45 Drives "Job Postings" page, and I created the new 45 Drives Blog starting from scratch, using WordPress. I also spent a considerable amount of time changing the design of certain pages on the 45 Drives site, to make them more user friendly, and to allow them to work as best as possible on mobile devices.

I mainly worked with on the 45 Drives site using the HTML and CSS languages, while using JavaScript for certain tasks, such as working with secondary navigational structures on certain pages. Additionally, I worked on 45 Drives' sitemap using XML.
